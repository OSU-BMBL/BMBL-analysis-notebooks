## Gotchas & Warnings

1. **Needs good reference**: Garbage in = garbage out - reference quality matters
2. **Cell type resolution**: Too many cell types = poor accuracy
3. **Rare cell types**: May not be detected if <1% in reference
4. **Mixture assumption**: Assumes each spot has multiple cell types

## Common Errors & Solutions

### Error: "Coordinate mismatch"
**Cause**: Image and data coordinates don't align  
**Solution**: Check `scale.factors` in spatial object

### Error: "Memory error"
**Cause**: High-resolution images  
**Solution**: Downsample images or use cropped regions
