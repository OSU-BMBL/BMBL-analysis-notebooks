
```r

```

```r
# Add disease-specific fields
metadata$disease_status <- c("control", "treatment")

```

## Gotchas & Warnings

1. **File Size Limits**: GEO has 2GB limit per file for direct upload - larger files need FTP
2. **Metadata Completeness**: All samples must have complete metadata - missing fields will cause rejection
3. **Raw vs Processed**: Must submit raw data (FASTQ) OR processed counts - GEO prefers both
4. **Release Date**: Set embargo date carefully - cannot be changed after submission
