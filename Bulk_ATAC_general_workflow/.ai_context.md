## Gotchas & Warnings

1. **Peak consistency**: Use same peak set across all samples
2. **Batch effects**: Common in ATAC - include in design formula
3. **GC bias**: ATAC has GC bias - check normalization
4. **Peak annotation**: Use ChIPseeker or similar for annotation
