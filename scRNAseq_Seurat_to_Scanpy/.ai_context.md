## Quick Summary

Convert Seurat objects to Scanpy/AnnData format for Python analysis.

**Typical Runtime**: 1-2 hours  
**Memory Required**: 16GB RAM  
**Input**: Seurat object (.rds)  
**Output**: H5AD file compatible with Scanpy

## Common Modifications

### 1. Select Assays to Export
**File**: `seurat_to_scanpy.rmd` (line 40)

```r
# Export only RNA assay to reduce size
DefaultAssay(seurat_obj) <- "RNA"
seurat_obj <- DietSeurat(seurat_obj, assays = "RNA")
```

### 2. Preserve DimReducs
**File**: `seurat_to_scanpy.rmd` (line 55)

```r
# Keep UMAP and PCA
SaveH5Seurat(seurat_obj, 
  filename = "output.h5Seurat",
  overwrite = TRUE,
  verbose = TRUE
)
```

---

## Gotchas & Warnings

1. **Assay names**: Must be valid Python identifiers (no spaces)
2. **Large objects**: Split by sample if >50k cells
3. **DimReducs**: Some may not transfer - re-run in Python if needed
4. **Metadata types**: Factors become strings in Python

## Common Errors & Solutions

### Error: "Insufficient cells"
**Cause**: Rare cell types have too few cells  
**Solution**: Merge similar cell types or collect more data

### Error: "Memory allocation failed"
**Cause**: Large datasets  
**Solution**: Use sketch-based methods or increase RAM
