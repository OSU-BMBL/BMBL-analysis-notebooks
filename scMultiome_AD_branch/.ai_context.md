# AI Context: scMultiome AD Branch (Single-Nucleus Multi-Omics)

## Quick Summary

This workflow integrates single-nucleus RNA-seq (snRNA-seq) and ATAC-seq (snATAC-seq) data from human brain tissue samples, specifically designed for Alzheimer's Disease (AD) research. Using Seurat v4 and Signac, it performs joint multi-modal analysis through Weighted Nearest Neighbor (WNN) integration, enabling simultaneous analysis of gene expression and chromatin accessibility from the same nuclei.

**Analysis Type**: Single-nucleus multi-omics (RNA + ATAC) integration  
**Primary Tools**: Seurat, Signac, SCTransform, TF-IDF, LSI  
**Typical Runtime**: 2-4 hours  
**Memory Requirements**: 32GB minimum, 64GB recommended  
**Platform**: OSC HPC or Local workstation

---

## Data Flow

Data transformation pipeline from input to output:

```
snRNA-seq + snATAC-seq data (Cell Ranger ARC output)
    ↓ (snMulti-omics-integration.rmd)
Separate RNA and ATAC preprocessing
    ↓
Quality control for both modalities
    ↓
RNA: SCTransform normalization + PCA
    ↓
ATAC: TF-IDF + LSI dimensionality reduction
    ↓
Integration across multiple samples (if applicable)
    ↓
WNN (Weighted Nearest Neighbor) joint embedding
    ↓
Joint clustering and UMAP visualization
    ↓
Cell type annotation using marker genes
    ↓
Differential accessible peaks (DAPs) analysis
    ↓
Integrated multi-omic results
```

**Input Location**: `filtered_feature_bc_matrix.h5`, `atac_fragments.tsv.gz`  
**Output Location**: Seurat objects, CSV files of DAPs, visualizations  
**Intermediate Files**: 
- Preprocessed RNA and ATAC assays
- LSI embeddings for ATAC
- WNN joint embeddings

## Common Modifications

### 1. Changing QC Thresholds
**Task**: Adjust quality control cutoffs for RNA and ATAC  
**File**: `snMulti-omics-integration.rmd`  
**Line/Variable**: QC subsetting section (around lines 85-93)  
**Change**:
```r
# Current QC thresholds:
multi <- subset(
  x = multi,
  subset = nCount_peaks < 40000 &
    nCount_RNA < 75000 &
    nCount_peaks > 1000 &
    nCount_RNA > 1000 &
    nucleosome_signal < 2.5 &
    TSS.enrichment > 1
)

# More stringent:
multi <- subset(
  x = multi,
  subset = nCount_peaks < 30000 &
    nCount_RNA < 50000 &
    nCount_peaks > 2000 &
    nCount_RNA > 2000 &
    nucleosome_signal < 2.0 &
    TSS.enrichment > 2
)
```
**Impact**: Changes number of nuclei retained. Balance between quality and cell yield.  
**Testing**: Check violin plots before/after QC. Aim to retain >70% of nuclei.

### 3. Adjusting WNN Weights
**Task**: Change relative weighting of RNA vs ATAC in joint embedding  
**File**: `snMulti-omics-integration.rmd`  
**Line/Variable**: `FindMultiModalNeighbors()` call  
**Change**:
```r
# Default (balanced):
combined <- FindMultiModalNeighbors(
  combined,
  reduction.list = list("pca", "lsi"),
  dims.list = list(1:30, 2:30),
  modality.weight.name = "RNA.weight"
)

# RNA-weighted (more emphasis on gene expression):
# Reduce ATAC dimensions or increase RNA dims

# ATAC-weighted (more emphasis on chromatin):
# Increase LSI dimensions used
combined <- FindMultiModalNeighbors(
  combined,
  reduction.list = list("pca", "lsi"),
  dims.list = list(1:20, 2:40),  # More ATAC dims
  modality.weight.name = "RNA.weight"
)
```
**Impact**: Changes how much RNA vs ATAC influences clustering. RNA-weighted better for cell type ID, ATAC-weighted better for regulatory analysis.  
**Testing**: Compare clustering with different weights using known marker genes.

### 5. Finding Differential Accessible Peaks
**Task**: Identify cell-type-specific open chromatin regions  
**File**: `snMulti-omics-integration.rmd`

**Impact**: Identifies regulatory elements specific to cell types or conditions.  
**Testing**: Check that top peaks are near known marker genes for that cell type.

## Gotchas & Warnings

### ⚠️ Critical Issues
- **ATAC dims start at 2**: First LSI component captures sequencing depth, not biology
  **Solution**: Always use `dims.list = list(1:30, 2:30)` - skip LSI dimension 1
  **Why**: LSI component 1 correlates with library size, should be excluded

- **Fragment file must be tabix-indexed**: Signac requires indexed fragment files
  **Solution**: Ensure `.tbi` index file exists alongside `.gz` fragment file
  **Check**: `ls *fragments.tsv.gz*` should show both .gz and .gz.tbi files

### 🔶 Common Mistakes
- **Using cell-level QC for nuclei**: Nuclei have different characteristics than cells
  **Why**: Nuclei naturally have less RNA, different mitochondrial content
  **Fix**: Use nuclei-appropriate thresholds (provided in workflow)

- **Mismatch between RNA and ATAC barcodes**: Barcodes may differ between modalities
  **Why**: Cell Ranger ARC may append suffixes
  **Fix**: Check barcode formats match between RNA and ATAC assays

### 📋 Dependencies Between Steps
- Must load both RNA and ATAC data before QC
- Must run TF-IDF and LSI on ATAC before WNN integration
- Must annotate cell types before DAP analysis
- Must have fragment files accessible for ATAC analysis

## Common Errors & Solutions

### Error: "Cannot find fragment file index"
**Cause**: Missing .tbi index file  
**Solution**: Create index with tabix:
```bash
tabix -p bed atac_fragments.tsv.gz
```
**Prevention**: Cell Ranger ARC should create this automatically

### Error: "No cells passed QC filters"
**Cause**: Too stringent thresholds  
**Solution**: Relax QC cutoffs, especially for nuclei (lower RNA counts expected)  
**Prevention**: Check QC metrics distributions before applying filters

### Error: "LSI component 1 is library size"
**Cause**: Included LSI dim 1 in analysis  
**Solution**: Always start from dim 2: `dims.list = list(1:30, 2:30)`

### Error: "Different number of cells in RNA and ATAC"
**Cause**: Barcode mismatches between modalities  
**Solution**: Check barcode formats, ensure they match
```r
# Check overlap:
length(intersect(Cells(rna_assay), Cells(atac_assay)))
```

## Parameter Guide

### Key Parameters
| Parameter | Default | Range | Description |
|-----------|---------|-------|-------------|
| `nCount_RNA min` | 1000 | 500-2000 | Min RNA counts per nucleus |
| `nCount_peaks min` | 1000 | 500-2000 | Min ATAC counts per nucleus |
| `nucleosome_signal` | < 2.5 | 1.5-4 | Fragment length distribution |
| `TSS.enrichment` | > 1 | 1-5 | Transcription start site signal |
| `LSI dims` | 2:30 | 2-50 | ATAC dimensions for WNN |
| `PCA dims` | 1:30 | 1-50 | RNA dimensions for WNN |

### When to Change Parameters
- **Lower thresholds**: For low-quality or degraded samples
- **Higher thresholds**: For high-quality data, want only best nuclei
- **More LSI dims**: For complex regulatory landscapes
- **Fewer dims**: For simpler datasets, faster computation
