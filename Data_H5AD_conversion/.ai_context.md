## Quick Summary

Convert between Seurat (R) and AnnData/H5AD (Python) formats.

**Typical Runtime**: 30 minutes - 2 hours  
**Memory Required**: 16GB RAM  
**Input**: Seurat object (.rds) or H5AD file (.h5ad)  
**Output**: Converted format for cross-platform compatibility

## Common Modifications

### 1. Convert Specific Assays
**File**: `convert_h5ad.rmd` (line 55)

```r
# Convert only RNA assay (smaller file)
SaveH5Seurat(seurat_obj, filename = "output.h5Seurat", overwrite = TRUE)
Convert("output.h5Seurat", dest = "h5ad", overwrite = TRUE)

# Skip other assays
```

### 2. Handle Large Objects
**File**: `convert_h5ad.rmd` (line 70)

```r
# Split by sample before conversion
for (sample in unique(seurat_obj$sample)) {
  subset_obj <- subset(seurat_obj, sample == !!sample)
  SaveH5Seurat(subset_obj, filename = paste0(sample, ".h5Seurat"))
}
```

---

## Gotchas & Warnings

1. ** assay naming**: Seurat assay names must be valid Python identifiers (no spaces, special chars)
2. **Dimensionality reduction**: UMAP/tsne coordinates may need re-running in Python
3. **Sparse matrices**: Some conversions lose sparsity - check file sizes
4. **Batch conversion**: Memory accumulates - restart R between large conversions
