## Common Modifications

### 1. Change Sketch Size
**File**: `sketch_analysis.rmd` (line 50)

```r
# Larger sketch for better representation
sketch <- SketchData(object = seurat_obj,
  ncells = 50000,  # Default may be 20000
  method = "LeverageScore"
)
```

### 2. Different Method
**File**: `sketch_analysis.rmd` (line 45)

```r
# Uniform sampling vs LeverageScore
sketch <- SketchData(object = seurat_obj,
  method = "Uniform"
)
```

---

## Gotchas & Warnings

1. **Sketch size**: 20-50k cells usually sufficient for population structure
2. **LeverageScore**: Better preserves rare populations than uniform
3. **Projection**: Full dataset maps to sketch reference - may lose fine details
4. **Integration**: Sketch first, then integrate if needed

## Common Errors & Solutions

### Error: "Insufficient cells"
**Cause**: Rare cell types have too few cells  
**Solution**: Merge similar cell types or collect more data

### Error: "Memory allocation failed"
**Cause**: Large datasets  
**Solution**: Use sketch-based methods or increase RAM
