## Quick Summary

Co-accessibility analysis to find peak-gene connections.

**Typical Runtime**: 4-6 hours  
**Memory Required**: 32GB RAM  
**Input**: scATAC-seq data with cell types annotated  
**Output**: Cicero connections, co-accessibility scores

---

## Data Flow

```
Cell Data (cds)
    |
    v
Aggregate by Cell Type
    |
    v
Calculate Co-accessibility
    |
    v
Peak-Gene Connections
```

## Common Modifications

### 1. Change Window Size
**File**: `cicero_analysis.rmd` (line 70)

```r
# Larger window for distal connections
conns <- run_cicero(cds,

)
```

### 2. Custom Threshold
**File**: `cicero_analysis.rmd` (line 85)

```r
# Filter connections
significant_conns <- conns[conns$coaccess > 0.2, ]
```

---

## Gotchas & Warnings

1. **Requires monocle3**: Cicero is built on monocle3 infrastructure
2. **Pseudotime optional**: Can use cell types instead of pseudotime
3. **Coaccess range**: Scores range -1 to 1; >0.2 is generally significant
4. **Cell aggregation**: Need sufficient cells per group (>50 recommended)

## Common Errors & Solutions

### Error: "Out of memory"
**Cause**: ATAC data is large  
**Solution**: Reduce `maxCells` parameter or use sparse matrices

### Error: "No peaks found"
**Cause**: Low quality data or wrong genome  
**Solution**: Check fragment files and genome version
