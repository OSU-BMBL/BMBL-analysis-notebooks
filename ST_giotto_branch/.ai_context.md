## Common Modifications

### 1. Change HMRF Parameters
**File**: `giotto_analysis.rmd` (line 80)

```r
# More spatially coherent clusters
hmrf_results <- doHMRF(giotto_obj,
  betas = c(0, 5, 10),  # Spatial smoothness
  k = 8  # Number of domains
)
```

### 2. Custom Network
**File**: `giotto_analysis.rmd` (line 60)

```r
# Delaunay vs KNN network
giotto_obj <- createSpatialNetwork(giotto_obj, 
  method = "Delaunay",
  minimum_k = 4
)
```

## Gotchas & Warnings

1. **Different from Seurat**: Giotto has its own object structure
2. **Python backend**: Some functions use Python through reticulate
3. **Visualization**: Giotto has excellent spatial plotting functions
4. **HMRF can fail**: With small datasets (<100 spots), HMRF may not converge

## Common Errors & Solutions

### Error: "Coordinate mismatch"
**Cause**: Image and data coordinates don't align  
**Solution**: Check `scale.factors` in spatial object

### Error: "Memory error"
**Cause**: High-resolution images  
**Solution**: Downsample images or use cropped regions
