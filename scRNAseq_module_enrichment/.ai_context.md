### 1. Custom Module Lists
**File**: `module_scoring.rmd` (line 45)

```r
# Define custom modules
stress_genes <- c("HSPA1A", "DNAJB1", "HSP90AA1")
seurat_obj <- AddModuleScore(seurat_obj, 
  features = list(stress = stress_genes),
  name = "stress_score"
)
```

### 2. Batch Correct Scores
**File**: `module_scoring.rmd` (line 70)

# Account for batch effects
library(limma)

## Gotchas & Warnings

1. **Module size**: 10-100 genes ideal - too few = noisy, too many = diluted
2. **Control genes**: Seurat picks control genes automatically - check they're appropriate
3. **Interpretation**: Scores are relative - compare within dataset
4. **Multiple modules**: Can score many modules, but check for redundancy

## Common Errors & Solutions

### Error: "Insufficient cells"
**Cause**: Rare cell types have too few cells  
**Solution**: Merge similar cell types or collect more data

### Error: "Memory allocation failed"
**Cause**: Large datasets  
**Solution**: Use sketch-based methods or increase RAM
