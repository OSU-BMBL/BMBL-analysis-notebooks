**Purpose**: Predict gene expression from ATAC accessibility

## Quick Summary

Predict gene expression from ATAC accessibility.

**Typical Runtime**: 3-4 hours  
**Memory Required**: 32GB RAM  
**Input**: scATAC-seq data with peak-to-gene links  
**Output**: Gene activity scores, imputed expression matrix

---

## Data Flow

```
Peak Matrix + Gene Annotations
    |
    v
Create Gene-Peak Links
    |
    v
Score Gene Activity
    |
    v
Activity Matrix (genes x cells)
```

## Common Modifications

### 1. Change Link Distance
**File**: `gene_activity.rmd` (line 65)

```r
# Extend link distance
links <- LinkPeaks(
  object = obj,
  peak.assay = "peaks",
  expression.assay = "RNA",
  distance = 200000  # 200kb (default 100kb)
)
```

### 2. Use Alternative Scoring
**File**: `gene_activity.rmd` (line 80)

```r
# Activity score based on nearby peaks
gene.activities <- GeneActivity(obj, 
  extend.upstream = 2000,
  extend.downstream = 2000
)
```

---

## Gotchas & Warnings

1. **Correlates with RNA**: Gene activity correlates ~0.3-0.5 with actual RNA
2. **Promoter bias**: Most predictive peaks are near TSS
3. **Multiome better**: True RNA from multiome > imputed activity
4. **Cell type specific**: Link peaks after clustering for better accuracy

## Common Errors & Solutions

### Error: "Out of memory"
**Cause**: ATAC data is large  
**Solution**: Reduce `maxCells` parameter or use sparse matrices

### Error: "No peaks found"
**Cause**: Low quality data or wrong genome  
**Solution**: Check fragment files and genome version
