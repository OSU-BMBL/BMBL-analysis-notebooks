## Quick Summary

Topic modeling for scATAC-seq using cisTopic.

**Typical Runtime**: 4-6 hours  
**Memory Required**: 64GB RAM  
**Input**: Binary accessibility matrix (peaks x cells)  
**Output**: Topic scores, regulatory topics, region-topic probabilities

## Gotchas & Warnings

1. **Binary input required**: cisTopic needs binary matrix (0/1), not counts
2. **Topic selection**: Use log-likelihood to find optimal topic number
3. **Python dependency**: cisTopic uses reticulate to call Python LDA
4. **Interpretation**: Topics = regulatory programs (not necessarily cell types)

## Common Errors & Solutions

### Error: "Out of memory"
**Cause**: ATAC data is large  
**Solution**: Reduce `maxCells` parameter or use sparse matrices

### Error: "No peaks found"
**Cause**: Low quality data or wrong genome  
**Solution**: Check fragment files and genome version
