## Common Modifications

### 1. Change Resolution Factor
**File**: `bayesspace_analysis.rmd` (line 65)

```r
# Higher resolution = more subspots
enhanced <- spatialEnhance(sce, 
  q = 8,  # Number of clusters
  platform = "Visium",
  nrep = 10000,
  gamma = 2  # Smoothing parameter
)
```

### 2. Platform Selection
**File**: `bayesspace_analysis.rmd` (line 45)

```r
# For Slide-seq instead of Visium
enhanced <- spatialEnhance(sce, platform = "Slide-seq")
```

## Gotchas & Warnings

1. **Memory intensive**: 9x increase in spots requires substantial RAM
2. **MCMC iterations**: Default 10,000 iterations - reduce for testing
3. **Cluster number**: Must specify q (clusters) before enhancement
4. **Platform matters**: Results differ between Visium and Slide-seq

## Common Errors & Solutions

### Error: "Coordinate mismatch"
**Cause**: Image and data coordinates don't align  
**Solution**: Check `scale.factors` in spatial object

### Error: "Memory error"
**Cause**: High-resolution images  
**Solution**: Downsample images or use cropped regions
