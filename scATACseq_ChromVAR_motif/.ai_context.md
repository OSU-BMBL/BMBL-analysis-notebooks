**Typical Runtime**: 4-6 hours  
**Memory Required**: 32GB RAM

## Gotchas & Warnings

1. **Requires peaks**: Must have peak calls before running chromVAR
2. **Motif database**: Default is JASPAR - can use custom PWMs
3. **Bias correction**: chromVAR corrects for GC bias - don't pre-filter
4. **Memory**: Large peak sets (>200k) require substantial memory

## Common Errors & Solutions

### Error: "Out of memory"
**Cause**: ATAC data is large  
**Solution**: Reduce `maxCells` parameter or use sparse matrices

### Error: "No peaks found"
**Cause**: Low quality data or wrong genome  
**Solution**: Check fragment files and genome version
